# 🃏 Card UI Project (HTML & CSS)

This is a simple and clean Card UI project built using **HTML and CSS**.  
It demonstrates a modern card-based layout with images, tags, text content, and buttons.

---

## 📸 Preview

Each card contains:
- An image
- Category tags (e.g. Nature, Lake)
- A title
- A short description
- A "Read More" button

---

## 🚀 Features

- Responsive card layout using **CSS Flexbox**
- Reusable card components
- Clean and minimal UI design
- Styled buttons and tags
- Simple and beginner-friendly structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
